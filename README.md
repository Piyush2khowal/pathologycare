# 🩺 Pathology Care Website

A modern, responsive pathology laboratory website built using **HTML**, **CSS**, and **JavaScript**. This project is designed to provide users with an easy and informative experience for exploring pathology services, health packages, and contact information.

## 🌐 Live Demo

https://pathologycare.netlify.app/

## ✨ Features

* Responsive design for all devices
* Modern and user-friendly interface
* Information about pathology services
* Health checkup packages
* Contact and enquiry section
* Fast and clean frontend implementation

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

```text
├── index.html
├── css/
├── js/
├── images/
└── assets/
```

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/pathology-care.git
```

2. Open the project folder.

3. Open `index.html` in your browser or use the VS Code Live Server extension.
